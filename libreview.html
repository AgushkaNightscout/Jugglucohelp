Libreview

Juggluco can send glucose data and amounts to Libreview.

If you use Abbotts Libre app for Freestyle Libre 2 sensors, Libreview is based on the history data out of the memory of the sensor. For this you need to scan the sensor.

Juggluco uses the stream data received via Bluetooth to create these 15 minute points by averaging the stream values around each point. Now and then it also sends an individual stream value in the same way as individual scanned values are sent to Libreview by Abbott's Librelink app. Juggluco doesn't need to be constantly connected to the internet for all data to be sent to Libreview. Like the Librelnk app, it sends data from the last 89 days to Libreview. What is necessary is that Juggluco is constantly connected to a Freestyle Libre 2 sensor via Bluetooth. For Freestyle Libre 3 sensors the 5 minutes history data is used, so that all previous values are sent to Libreview.

To make use of this service, you need to create a Libreview account at https://libreview.com and specify its E-Mail address and password here. Don't send data from the same period multiple times to the same Libreview account (for example from the Librelink app and Juggluco). You can delete existing devices in Libreview in Account Settings->My devices or you can take a new Libreview account.

To turn it on, you also need to check "Send to libreview".

Juggluco will send new data to Libreview every time you switch away from Juggluco when it wasn't done in the previous 20 minutes. You can also manually send it to Libreview by pressing "Send now".

Beside glucose values, amounts can also be sent to Libreview. For this you need to touch "Send amounts" and specify what each label means.

Starting with Juggluco 4.1.0 glucose values from FreeStyle Libre 3 sensors are also send to LibreView.

Libre3 immediate: Send glucose values to Libreview immediately when they arrive from Libre 3 sensors. This is only useful in combination with LibreLinkUp. To make use of it you need to add a LibrelinkUp connection to this Libreview account in one of Abbott's Libre apps. Sometimes the value is not immediately visible, because display of LibreLinkUp isn't redrawn automatically. LibreLinkUp can not be used with Libre 2 sensors: one cannot plausibly claim to scan every minute the whole day long. Instead, make use of a mirror connection or Nightscout.

Resend data: Start anew; resend data of the last 90 days to Libreview. You need to press this when switching to a new Libreview account.

Result of the last time trying to send to Libreview is displayed below the "Get Account ID" button.

Because the values sent to Libreview are first averaged over 15 minutes and then analyzed, there can be small differences between the values shown in Juggluco's statistics screen and Libreview. For example Libreview says 100% of the time active, while Juggluco's statistics screen says 96.0%. Also some high and low values will be averaged away, so that Libreview gives a somewhat higher time in target than the statistics screen of Juggluco. Because the real history values are less extreme than the average of the stream values around a history value, this difference will be larger when Libreview is based on scanned values.

Juggluco can be connected with multiple sensors at the same time, while Librelink only handles one sensor at a time. Juggluco also sends data from only one sensor at a time to Libreview. It does this by first sending the data from one sensor and only after no data is available from that sensor, it sends the data from the following sensor. When Juggluco is switched to a following sensor, Juggluco will not turn back to a previous one, to imitate the one sensor policy of Librelink. This means that when you run multiple sensors at a time and a later added one ends earlier than an earlier added sensor, this earlier sensor will send no data anymore to Libreview.

Get Account ID: If you only want to receive the LibreView account ID, but not send data to Libreview, you can fill in your E-Mail address and Password and press on "Get Account ID". On the same line before this button, the Account ID will be shown (this can take some time and the screen is not refreshed automatically). Retrieving the Account ID from the LibreView server makes it possible to use a FreeStyle Libre 3 sensor that was already used with Abbotts Libre 3 app with this LibreView account. Starting the sensor with Juggluco is reported not to work.

WARNING: The Freestyle Libre 3 sensor should be activated with this account.
