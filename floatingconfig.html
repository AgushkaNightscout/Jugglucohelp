Floating glucose

Set font size and foreground and background color of the floating glucose. You can also specify Transparent for the background, making it invisible.

After typing the font size, you have to press the “Enter” or “Done” button of the soft keyboard.

You can turn on and off Floating glucose with the “Floating glucose” checkbox in settings. You can also turn off “Floating glucose” by long pressing on the floating glucose value. For example when you have set the font size so large that you are not able anymore to type a smaller font size on the soft keyboard.
