Configuração de Kerfstok

A fonte de Kerfstok pode ser baixada em http://jkaltes.byethost16.com/Kerfstok/Kerfstok-source.html ). Os utilizadores são livres para modificá-lo. Cada aplicativo de relógio tem um identificador exclusivo. Se o identificador for diferente do padrão, o Juggluco precisa conhecê-lo para se comunicar com o aplicativo do relógio. Para especificar um identificador diferente, pressione ID e digite a string hexadecimal de 32 caracteres. Juggluco em um dispositivo só pode ser conectado a um aplicativo de relógio apenas em um relógio. As alterações levarão à perda de dados.

Atalhos

Os atalhos são enviados para o aplicativo de relógio Garmin Kerfstok para inserir facilmente determinados valores.

Modo escuro

Especifique se Kerfstok deve ser branco sobre preto em vez de preto sobre branco.
