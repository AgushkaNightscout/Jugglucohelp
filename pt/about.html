Sobre

Juggluco é escrito por Jaap Korthals Altes

Além das bibliotecas padrão do Android, são utilizadas as seguintes:

  • Comandos NFC e Bluetooth específicos para sensores Abbott FreeStyle Libre e algoritmos de interpretação específicos para sensores FreeStyle Libre. Todos são indocumentados;

  • Para se comunicar com os relógios Garmin, é feito uso do Connect IQ Mobile SDK ;

  • Os gráficos são criados com NanoVG ;

  • A função de espelhar usa criptografia autenticada Ascon ;

  • A comunicação com sensores Libre3 usa TinyCrypt ;

  • O seletor de cores é uma adaptação do AmbilWarna .

Juggluco foi traduzido para o italiano por Matteo Gabriele .

Juggluco foi traduzido para o português por Rui Pinheiro.
