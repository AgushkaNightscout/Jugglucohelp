Lembretes

Os lembretes o recordam se não inseriu uma determinada quantia em um determinado intervalo de tempo.

Ao pressionar NOVO , pode adicionar um novo lembrete.

Especifique a hora inicial e final do período de tempo, dentro do qual deve inserir o valor especificado na etiqueta especificada.

Se o horário de término for alcançado, o programa verificará se inseriu pelo menos esse valor com a etiqueta especificada após o horário de início. Se não se registou, uma notificação é exibida e um toque é reproduzido.

Por exemplo, se especificar:

Levemir 6.0

21h30 - 23h55

Será lembrado após as 23h55 se não tiver especificado pelo menos 6 unidades de Levemir após as 21h30 do mesmo dia.

Lembretes já especificados são mostrados em uma lista. Toque para editar.

Se tudo correr bem, os lembretes também funcionam após a reinicialização do Android sem que o utilizador precise iniciar o Juggluco.



Em alguns dispositivos Huawei, precisa especificar que permite que um programa seja iniciado na inicialização. Isso é feito em Configurações do Android -> Bateria -> Inicialização do aplicativo. Aqui seleciona Manualmente gerenciado para Juggluco para que Auto-Launch e Run in Background estejam ativados. Isso também é necessário para que os alarmes de perda de sinal funcionem após a reinicialização sem que o usuário inicie o Juggluco.
