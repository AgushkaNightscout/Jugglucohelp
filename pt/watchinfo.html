Relógio

Atualmente Juggluco tem quatro maneiras de obter valores de glicose em seu relógio.

Notificar

Se o Notificar estiver ativado no menu central esquerdo, uma notificação do Android será criada toda vez que um novo valor de glicose chegar. A maioria dos smartwatches pode ser configurada para receber essas notificações. Funciona com relógios Garmin. Alguns aplicativos complementares de outros relógios não enviam essa notificação para o relógio, provavelmente porque é uma modificação de uma notificação que está constantemente presente, em vez de criar uma nova notificação.

Kerfstok

Kerfstok é um aplicativo de relógio pelo qual pode inserir números (como doses de insulina) , mas também exibe o valor atual da glicose. Assim que chega um novo valor de glicose, Juggluco o envia para Kerfstok. Portanto, Kerfstok exibe sempre o valor de glicose mais recente. Kerfstok também pode registar todas as atividades suportadas pelo Garmin e pode exibir velocidade e distância. Veja Relógio->Kerfstok Estado-> Ajuda  https://www.juggluco.nl/Kerfstok

Aplicações de relógio xDrip

Juggluco incorpora um servidor web xDrip. Isso possibilita o uso direto de aplicativos de relógio xDrip com Juggluco. O xDrip exibe um novo valor de glicose apenas a cada 5 minutos, enquanto o Juggluco exibe um novo valor de glicose a cada minuto. Juggluco também envia para os aplicativos de relógio xDrip o valor de glicose mais recente. Os valores de glicose são enviados apenas para os relógios se o aplicativo do relógio os solicitar, portanto, a idade dos valores exibidos depende de quando o aplicativo do relógio os solicita. Por exemplo, mostradores de relógio e campos de dados em relógios Garmin solicitam apenas um valor de glicose a cada 5 minutos. Através do xDrip, isso geralmente permite valores de glicose de 10 minutos. Quando conectados diretamente com Juggluco, esses mostradores de relógio mostram valores de glicose de até 6 minutos. Em Garmin, se não puder usar Kerfstok, tem as seguintes opções para receber regularmente um valor recente de glicose. Um widget xDrip ( https://apps.garmin.com/en-US/apps/73115e04-dc9f-4765-ad88-e7ae283ce995 ) também pode ser usado durante a gravação de uma atividade. Portanto, se tiver uma área livre, poderá alternar para este widget xDrip enquanto grava uma atividade com um aplicativo desportivo Garmin padrão. Há também um aplicativo desportivo xDrip que atualiza seu valor de glicose regularmente: https://apps.garmin.com/en-US/apps/aaf6533b-bca1-4365-9131-9f882f6f148c . Pode tentar iniciar em um momento, para que ele solicite um valor de glicose logo após estar disponível. Campos de dados e mostradores de relógio seriam a solução ideal, mas as restrições impostas pela Garmin os tornam de pouco valor.

De acordo com um utilizador, existe também um mostrador de relógio para relógios Fitbit que funciona com o servidor web xDrip: https://glancewatchface.com

Para obter mais informações sobre a interface do servidor da Web, consulte https://www.juggluco.nl/Juggluco/addwatch.html

Wear OS Juggluco

Existe uma versão do Juggluco adaptada para o Wear OS. A exibição é adaptada para o ecrã menor e um mostrador de relógio é adicionado. O mostrador do relógio pode ser instalado como um mostrador normal do Wear OS e exibe ao lado da hora também o valor atual de glicose recebido via Bluetooth.
