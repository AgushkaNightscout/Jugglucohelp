Selecione como os valores em Juggluco inseridos em uma etiqueta específica devem ser enviados para o Libreview. É possível enviar mais de um tipo de valor com a mesma etiqueta Libreview. Por exemplo, se classificar alimentos com carboidratos e glicose pura em etiquetas separadas no Juggluco, poderá especificar para ambos que eles devem ser enviados como carboidratos para o Libreview.

Para Carboidrato, é possível multiplicar a quantidade dada em Juggluco por um determinado número para que possa transformar outras unidades em gramas. Se contar no Juggluco porções de 10 gramas, pode inserir aqui 10, para que as gramas sejam enviadas para o Libreview.

Comentários significa que o valor é enviado para o Libreview como se você tivesse adicionado comentários no Librelink. Se especificar comentários aqui, a etiqueta mais o número inserido para essa etiqueta será enviado. Por exemplo: " Bicicleta 50 ", se você tivesse uma etiqueta " Bicicleta " e inserisse " 50 " sob essa etiqueta. " Bicicleta 50 " será mostrado sob o gráfico do " Registro diário " no Libreview no horário especificado.
