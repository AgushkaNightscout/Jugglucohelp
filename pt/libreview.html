Libreview

Juggluco pode enviar dados e quantidades de glicose para o Libreview.

Se usar o aplicativo Abbott Libre para sensores Freestyle Libre 2, o Libreview será baseado nos dados históricos da memória do sensor. Para isso, precisa escanear o sensor.

Juggluco usa os dados de stream recebidos via Bluetooth para criar esses pontos de 15 minutos calculando a média dos valores de stream em torno de cada ponto. De vez em quando, ele também envia um valor de stream individual da mesma forma que os valores digitalizados individuais são enviados para o Libreview pelo aplicativo Librelink da Abbott.

Para fazer uso deste serviço, precisa criar uma conta Libreview em https://libreview.com e especificar seu endereço de e-mail e senha. Não envie dados do mesmo período várias vezes para a mesma conta Libreview (por exemplo, do aplicativo Librelink e Juggluco).  pode excluir dispositivos existentes no Libreview em Configurações da conta->Meus dispositivos ou pode criar uma nova conta do Libreview.

Para ativá-lo, também precisa marcar " Enviar para libreview " .

O Juggluco enviará novos dados para o Libreview toda vez que sair do Juggluco quando isso não tiver sido feito nos 20 minutos anteriores.  Também pode enviá-lo manualmente para o Libreview pressionando " Enviar agora " .

Além dos valores de glicose, os valores também podem ser enviados para o Libreview. Para isso precisa tocar em " Enviar valores " e especificar o que significa cada etiqueta.

A partir do Juggluco 4.1.0, os valores de glicose dos sensores FreeStyle Libre 3 também são enviados para o LibreView.

Reenviar dados : Comece de novo; reenvie os dados dos últimos 90 dias para o Libreview. precisa pressionar isso ao mudar para uma nova conta do Libreview.

O resultado da última tentativa de envio para o Libreview é exibido abaixo do botão "Obter ID da conta".

Como os valores enviados para o Libreview são primeiro calculados em 15 minutos e depois analisados, pode haver pequenas diferenças entre os valores mostrados na tela de estatísticas do Juggluco e o Libreview. Por exemplo, o Libreview diz 100% do tempo ativo, enquanto a tela de estatísticas do Juggluco diz 96,0%. Além disso, alguns valores altos e baixos serão calculados, de modo que o Libreview forneça um tempo um pouco mais alto no alvo do que a tela de estatísticas do Juggluco. Como os valores reais do histórico são menos extremos do que a média dos valores de stream em torno de um valor histórico, essa diferença será maior quando o Libreview for baseado em valores digitalizados.

O Juggluco pode ser conetado a vários sensores ao mesmo tempo, enquanto o Librelink lida apenas com um sensor por vez. O Juggluco também envia dados de apenas um sensor por vez para o Libreview. Ele faz isso enviando primeiro os dados de um sensor e somente depois que nenhum dado estiver disponível desse sensor, ele envia os dados do sensor seguinte.

Obter ID da conta : Se deseja apenas receber o ID da conta do LibreView, mas não enviar dados para o Libreview, preencha seu endereço de e-mail e senha e pressione " Obter ID da conta ". Na mesma linha antes deste botão, o ID da conta será mostrado (isso pode levar algum tempo e a tela não é atualizada automaticamente). A recuperação da ID da conta do servidor LibreView possibilita o uso de um sensor FreeStyle Libre 3 que já foi usado com o aplicativo Abbott Libre 3 com esta conta LibreView. É relatado que iniciar o sensor com Juggluco não funciona.

AVISO : O sensor Freestyle Libre 3 deve ser ativado com esta conta. Quando mudar posteriormente para uma conta no aplicativo Libre 3, o scan do sensor resultará em um erro.
