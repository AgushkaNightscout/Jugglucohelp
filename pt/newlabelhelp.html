Etiqueta:

Nome da etiqueta a ser usada para inserir números. Pode adicionar um número vinculado a uma determinada data e hora usando "Novo valor" no menu do meio à esquerda. Esses números podem ser exibidos no gráfico de glicose naquela posição de tempo ou em uma lista.

Arredondar para:

Usado no aplicativo de relógio, Kerfstok. Os números inseridos sob essa etiqueta são arredondados para esse número.

0= precisão total, 0,5 = arredondar para múltiplos de 0,5 etc.

Peso:

Quando definido como 0, os números são exibidos em uma determinada altura no gráfico independente de seu valor. Se o peso for maior que 0, cada valor inserido sob esta etiqueta é multiplicado por este peso e exibido como um quadrado nas mesmas coordenadas do gráfico de glicose. Por exemplo, pode definir o peso como 1 para exibir as medições de glicose por picada no dedo da mesma forma que as medições do sensor de glicose.
