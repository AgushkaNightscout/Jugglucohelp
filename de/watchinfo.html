Uhren

Derzeit bietet Juggluco vier Möglichkeiten, um Glukosewerte auf Ihrer Uhr anzuzeigen.

Benachrichtigen

Wenn Benachrichtigen aktiviert ist, wird jedes Mal eine Android-Benachrichtigung erstellt, wenn ein neuer Glukosewert eintrifft. Bei einigen Uhren werden Benachrichtigungen nur an die Uhr gesendet, wenn eine neue Benachrichtigung erstellt wird, für sie müssen Sie Separat aktivieren . Für diese Uhren müssen Sie auch Separat einschalten, um Alarmbenachrichtigungen auf der Uhr zu erhalten.

Watchdrip

Durch das Einschalten sendet Juggluco Glukosewerte an Watchdrip+ . Watchdrip+ sendet Zifferblätter mit Glukosewerten an MiBand/Amazfit-Uhren.

GadgetBridge

GadgetBridge (Wiki) ermöglicht die Anzeige von Wetterinformationen auf MiBand/Amazfit-Uhren. Wenn die Gadgetbridge-Option aktiviert ist, sendet Juggluco Glukosewerte unter dem Vorwand, dass es sich um Wetterinformationen handelt, an GadgetBridge. Dies hatte gegenüber Watchdrip den Vorteil, dass es deutlich schneller ist, hat aber den Nachteil, dass weniger Informationen angezeigt werden. Juggluco trägt den Glukosewert und einen Richtungspfeil in den Wetterort ein. Dieser Ort wird auf der Uhr unter Wetter angezeigt. Ich habe kein Zifferblatt gefunden, das diesen Ort anzeigt. Der gerundete Glukosewert wird in die aktuelle Temperatur übernommen. Nur für mmol/L: Der Wert vor dem Punkt wird in die maximale Temperatur eingetragen und der Wert nach dem Punkt in die minimale Temperatur. Die Werte von mg/dL sind für Temperaturen zu groß, deshalb wird die letzte Ziffer bei der Minimaltemperatur und die Zahlen vor der letzten Ziffer bei der Maximaltemperatur eingetragen. Somit ergibt sich aus 106 mg/dL: maximale Temperatur 10, minimale Temperatur 6.

Die Richtung der Änderung bestimmt das Wettersymbol: Schnee bedeutet fallende Glukose, Regen bedeutet steigende Glukose. Es wird auch mit die aktuelle Temperatur eingegeben. Größer als Null bedeutet steigender Glukosewert, kleiner als Null bedeutet fallender Glukosewert.

Kerfstok

Kerfstok ist eine Uhren-App, mit der Sie Zahlen (wie Insulindosen) eingeben können, aber auch den aktuellen Glukosewert anzeigen. Sobald ein neuer Glukosewert eintrifft, sendet Juggluco diesen sofort nach Kerfstok. So zeigt Kerfstok immer den aktuellsten Glukosewert an. Kerfstok kann auch alle von Garmin unterstützten Aktivitäten aufzeichnen und kann Geschwindigkeit und Distanz anzeigen. Siehe Watch->Kerfstok Status-> Help und https://www.juggluco.nl/Kerfstok

Broadcast

Apps können Glukosewerte von einer der von Juggluco gesendeten Broadcasts empfangen und diese Glukosewerte an eine Uhr senden. Beispielsweise kann G-Watch (TizenOS oder WearOS) Glukosewerte von Glucodata Broadcast empfangen (siehe linkes Menü -> Einstellungen).

xDrip Watch-Apps

Juggluco enthält einen xDrip-Webserver. Dadurch ist es möglich, xDrip-Watch-Apps direkt mit Juggluco zu verwenden. xDrip zeigt nur alle 5 Minuten einen neuen Glukosewert an, während Juggluco jede Minute einen neuen Glukosewert anzeigt. Juggluco sendet auch den letzten Glukosewert an die xDrip-Uhren-Apps. Die Glukosewerte werden nur dann an die Uhren gesendet, wenn die Uhren-App danach fragt, also wie alt die angezeigten Werte sind, hängt davon ab, wann die Uhren-App danach fragt. Zum Beispiel fragen Zifferblätter und Datenfelder auf Garmin-Uhren nur alle 5 Minuten nach einem Glukosewert. Über xDrip lassen sich diese oft auf 10 Minuten alte Glukosewerte zurückführen. Bei direkter Verbindung mit Juggluco zeigen diese Zifferblätter bis zu 6 Minuten alte Glukosewerte an. Unter Garmin, wenn Sie Kerfstok nicht verwenden können, haben Sie die folgenden Optionen, um regelmäßig einen aktuellen Glukosewert zu erhalten. Ein xDrip-Widget ( https://apps.garmin.com/de-DE/apps/73115e04-dc9f-4765-ad88-e7ae283ce995 ) kann auch während der Aufzeichnung einer Aktivität verwendet werden. Wenn Sie also eine Hand frei haben, können Sie zu diesem xDrip-Widget wechseln, während Sie eine Aktivität mit einer Standard-Garmin-Sport-App aufzeichnen. Es gibt auch eine xDrip-Sport-App, die ihren Glukosewert regelmäßig aktualisiert: https://apps.garmin.com/de-DE/apps/aaf6533b-bca1-4365-9131-9f882f6f148c . Sie können versuchen, in einem Moment zu beginnen, so dass ein Glukosewert angefordert wird, sobald er verfügbar ist. Datenfelder und Zifferblätter wären die ideale Lösung, aber die von Garmin auferlegten Einschränkungen machen sie wenig wert.

Laut einem Benutzer gibt es auch ein Watchface für Fitbit- Uhren, das mit dem xDrip-Webserver funktioniert: https://glancewatchface.com

Weitere Informationen zur Webserver-Schnittstelle finden Sie unter https://www.juggluco.nl/Juggluco/webserver.html

local only : akzeptiert nur Verbindungen von localhost (127.0.0.1). Wenn Sie es ausschalten, können auch andere Hosts auf den Server zugreifen. Wenn Sie Ihre Firewall nicht so konfigurieren, dass sie auf Port 17580 umleitet, sind dies nur Hosts in Ihrem Heimnetzwerk. Dieser Server verwendet nur http. Ich habe auch eine https-Version erstellt, aber das scheint ohne ein von einer Zertifizierungsstelle signiertes Zertifikat für Ihren Hostnamen nutzlos zu sein. Also habe ich es weggelassen.

Juggluco for WearOS

Es gibt eine Version von Juggluco, die auf Wear OS portiert wurde. Die Anzeige wird an den kleineren Bildschirm angepasst und ein Watchface hinzugefügt. Das Watchface lässt sich wie ein gewöhnliches Wear OS Watchface installieren und zeigt neben der Uhrzeit auch den per Bluetooth empfangenen aktuellen Glukosewert an.
