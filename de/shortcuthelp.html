Shortcuts können in Kerfstok verwendet werden.

In der Zahleneingabemaske auf der Uhr gelangt man zum Shortcuts-Screen (auf meiner Vivoactive 3 Uhr durch Wischen nach links). Hier sehen Sie eine Liste von Etikette; Durch Drücken auf einen von ihnen wird ein entsprechender Wert in den Zahlenbildschirm eingefügt. Die Werte können Zahlen oder arithmetische Operatoren in Kombination mit Zahlen sein, zB *0.5 um mit 0.5 zu multiplizieren.

Hier im Einstellungsmenü können Sie diese Shortcuts erstellen, entfernen und bearbeiten.

Auf der linken Seite wird eine Liste bestehender Shortcuts angezeigt. Nachdem Sie eine davon berührt haben, erscheint eine Anzeige, in der Sie die Shortcut löschen oder die Bezeichnung oder den Wert ändern können.

Mit Neu wird eine neue Shortcut erstellt .

Änderungen werden erst nach Drücken der Schaltfläche Speichern gespeichert.
