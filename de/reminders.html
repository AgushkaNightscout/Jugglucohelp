Erinnerungen

Erinnerungen erinnern Sie daran, wenn Sie innerhalb eines bestimmten Zeitintervalls einen bestimmte Menge nicht eingegeben haben.

Durch Drücken von NEU können Sie eine neue Erinnerung hinzufügen.

Geben Sie Beginn und Ende des Zeitraums an, innerhalb dessen Sie den angegebene Menge unter dem angegebenen Etikett eintragen sollen.

Wenn die Endzeit erreicht ist, prüft das Programm, ob Sie nach der Startzeit mindestens diese Menge mit dem angegebenen Etikett eingegeben haben. Wenn Sie sich nicht registriert haben, wird eine Benachrichtigung angezeigt und ein Klingelton abgespielt.

Wenn Sie beispielsweise angeben:

Levemir 6.0

21:30 - 23:55

Sie werden nach 23:55 Uhr erinnert, wenn Sie nicht mindestens 6 Einheiten Levemir nach 21:30 Uhr desselben Tages angegeben haben.

Bereits festgelegte Erinnerungen werden in einer Liste angezeigt. Zum Bearbeiten berühren.

Wenn alles gut geht, funktionieren Erinnerungen auch nach einem Neustart von Android, ohne dass der Benutzer Juggluco starten muss.

Auf einigen Huawei-Geräten müssen Sie angeben, dass Sie zulassen, dass ein Programm beim Booten gestartet wird. Dies geschieht unter Android-Einstellungen -> Akku -> App-Start. Hier wählen Sie Manuell verwaltet für Juggluco aus, damit Auto-Launch und Run in Background aktiviert sind. Dies ist auch notwendig, damit Signalverlustalarme nach einem Neustart funktionieren, ohne dass der Benutzer Juggluco startet.
