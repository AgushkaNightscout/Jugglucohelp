Канфігурацыя Kerfstok

Зыходны код Kerfstok можна загрузіць з https://www.juggluco.nl/Kerfstok/Kerfstok-source.html). Карыстальнікі могуць змяняць яго. Кожнае прыкладанне для гадзінніка мае унікальны ідэнтыфікатар. Калі ідэнтыфікатар адрозніваецца ад ідэнтыфікатара па змаўчанні, Juggluco павінен ведаць яго, каб мець зносіны з дадаткам гадзінніка. Каб указаць іншы ідэнтыфікатар, націсніце ID і ўвядзіце шаснаццатковы радок з 32 сімвалаў. Juggluco на адной прыладзе можна падключыць толькі да адной праграмы для гадзінніка на адным гадзінніку. Змены прывядуць да страты дадзеных.

Ярлыкі

Ярлыкі адпраўляюцца ў дадатак для гадзіннікаў Garmin Kerfstok, каб лёгка ўставіць пэўныя значэнні.
