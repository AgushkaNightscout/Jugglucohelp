À propos

Juggluco est écrit par Jaap Korthals Altes

Outre les bibliothèques Android standard, il est fait usage des éléments suivants :

  • Commandes NFC et Bluetooth spécifiques aux capteurs Abbott FreeStyle Libre et algorithmes d'interprétation spécifiques aux capteurs FreeStyle Libre. Tous sont sans documentation;

  • Pour communiquer avec les montres Garmin, il est fait usage du SDK Connect IQ Mobile ;

  • Les graphiques sont créés avec NanoVG ;

  • La fonction miroir utilise le cryptage authentifié Ascon ;

  • La communication avec les capteurs Libre3 utilise TinyCrypt ;

  • Le sélecteur de couleurs est une adaptation d' AmbilWarna ;

  • La prise en charge de NFC NovoPens est une modification du code extrait de xDrip.
