Sélectionnez comment les quantités saisies dans Juggluco sous une étiquette particulière doivent être envoyées à Libreview. Il est possible d'envoyer plus d'un type de montant avec la même étiquette Libreview. Par exemple, si vous notez les aliments glucides et glucose pur sous des étiquettes distinctes dans Juggluco, vous pouvez spécifier pour les deux qu'ils doivent être envoyés en tant que glucides à Libreview.

Pour les glucides, il est possible de multiplier la quantité donnée dans le Juggluco par un certain nombre afin de pouvoir transformer d'autres unités en grammes. Si vous comptez dans Juggluco des portions de 10 grammes, vous pouvez entrer ici 10, afin que les grammes soient envoyés à Libreview.

Commentaires signifie que le montant est envoyé à Libreview comme si vous aviez ajouté des commentaires dans Librelink. Si vous spécifiez des commentaires ici, l'étiquette plus le numéro saisi pour cette étiquette sont envoyés. Par exemple : " Vélo 50 ", si vous aviez un libellé " Vélo " et que vous avez entré " 50 " sous ce libellé. " Vélo 50 " sera affiché sous le graphique du " Journal quotidien " dans Libreview à l'heure spécifiée.
