Etiquette:

Nom de l'étiquette à utiliser pour saisir des quantités. Vous pouvez ajouter une quantité liée à une certaine date et heure en utilisant "Nouvelle quantité" dans le menu du milieu à gauche. Ces nombres peuvent être affichés dans le graphique de glucose à cette position temporelle ou dans une liste.

Arrondir à:

Utilisé dans l'application de montre, Kerfstok. Les quantités saisies sous cette étiquette sont arrondies à ce nombre.

0 = précision totale ; 0,5 = arrondi aux multiples de 0,5 ; etc.

Coéfficient:

Lorsqu'il est défini sur 0, les nombres sont affichés à une certaine taille dans le graphique indépendamment de leur valeur.

Si le coéfficient est supérieur à 0, chaque quantité saisie sous cette étiquette est multipliée par ce coéfficient et affichée sous forme de carré dans les mêmes coordonnées que le graphique de glucose. Par exemple, vous pouvez régler le coéfficient sur 1 pour afficher les mesures de glucose par piqûre au doigt de la même manière que les mesures du capteur de glucose.
