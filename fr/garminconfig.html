Configuration de Kerfstock

La source de Kerfstok peut être téléchargée à partir de https://www.juggluco.nl/Kerfstok/Kerfstok-source.html ). Les utilisateurs sont libres de le modifier.

Chaque application de montre a un identifiant unique. Si l'identifiant est différent de celui par défaut, Juggluco a besoin de le connaître pour communiquer avec l'application de la montre. Pour spécifier un identifiant différent, appuyez sur ID et saisissez la chaîne hexadécimale de 32 caractères.

Juggluco sur un appareil ne peut être connecté qu'à une seule application de montre sur une seule montre. Les modifications entraîneront une perte de données.

Raccourcis

Des raccourcis sont envoyés à l'application de montre Garmin Kerfstok pour insérer facilement certaines valeurs.

Mode sombre

Spécifiez si Kerfstok doit être blanc sur fond noir au lieu de noir sur fond blanc.
